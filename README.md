To install dependencies:
```sh
bun install
```

To run:
```sh
bun run dev
```

open http://localhost:3000

example query (enter in address bar): http://localhost:3000/?chatstr=what%20is%20the%20best%20way%20to%20spend%20my%20time%20this%20weekend?
(you can usually also enter this as just a sentence, the %20 is how spaces get automatically reformatted in the address bar)
result will (Eventually) show up in your browser, provided ollama is running and has the right image installed.
Also prints nice formatted output of the conversation to the console while running. 

No frontend as of yet.
